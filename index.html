Introduction

Jenkins Khan is a php tool that helps you to manage a Jenkins integration server.

With Jenkins Khan you can easily launch different jobs on your working branch and check the result of each job group by your branches.

Why

When it comes to working with many branches, Jenkins is not very user-friendly. It's kind a mess to check your jobs state on every branch. We need an overview.

How ?

Jenkins Khan uses Jenkins' API to retrieve the configuration, jobs, jobs parameters, view, status, etc. You just have to register your Jenkins Url in Jenkins Khan and enjoy the fun.

Technology stack

Jenkins Khan is a standalone LAMP project.
It is based on web common technologies :

  • symfony 1.x
  • jQuery
  • Twitter bootstrap
  • LESS
You can easily install it on your own server.

Licence

Features

  • Display branch by branch the result of each Jenkins' build
  • Fully based on Jenkins' API
    • No configuration (except your Jenkins' url)
    • All views, jobs, configuration are immediatly available
  • Relaunch build one by one
  • Relaunch every build on a build branch
  • Register build without launching them on Jenkins
  • User-friendly interface.

Requirements

  • PHP 5.2+
  • MySQL 5+
  • Jenkins 1.4+
  • Jenkins Git Plugin
